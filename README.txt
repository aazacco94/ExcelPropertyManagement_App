You can run the program by opening EPM_Program.xlsm and clicking the sign-in button.
All of the required username/password pairs are included that were given on the canvas page.
Files that are included in the main directory are: 
1.) EPM_Program.xlsm 
	- Contains all the VBA Code for the program
2.) ExcelPrpertyManagement.xlsx
	- Property spreadsheet containing all of the property information
	- Holds a template file for new properties
3.) EPMusers.accdb
	- Holds the user account information in the EPMusers table
